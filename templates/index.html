Overview

Searching for common terms such as 'Cancer' take a long time. With 100's of thousands of records to retrieve. However searching for 'Parkinsons' over the same time span yields 11,000 records. Typically, the Fetch request has to be done in batches, with the max batch_size being 10,000, hence many requests are made.

Tools Used

  • Flask lightweight web framework
  • Python requests library for making HTTP requests to NCBI
  • Graph generated using d3js
